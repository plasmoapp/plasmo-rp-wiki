# Кастомные крафты

## Прозрачные рамки

Чтобы создать прозрачные рамки — обложите зелье невидимости из дыхания дракона (0:45) обычными рамками

![](/assets/features/crafts/frame.png)

## Невидимые блоки света

Из одного оседающего зелья невидимости, и любых трёх светящихся блоков, можно сделать невидимые блоки света

![](/assets/features/crafts/light.png)
